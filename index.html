Introduction

The game has 3 levels. 1st level has 2 prompts, 2nd level has 3 prompts & 3rd level has 5 prompts. After every level you can see the results how correct you are. Prompt could have multiple keywords as answer and 1 keyword can be 1-3 words long. Game also registers partly correct answer for example if the prompt answer is "interview notes" and you insert "notes" then you got half a point.
